SouqNet Image Predictor

What is this?

Great question! This is an image prediction application using convolutional neural network technology, the coolest thing in computer science since the Internet.

How was this made?

We scraped some images of fashionable items from the UAE Souq.com website and trained a convolutional neural network. The libraries we used include Keras, TensorFlow and a lot of brain power. This web application is made using Flask.
